# **App Name**: SkillSphere AI

## Core Features:

- Secure Authentication System: Comprehensive user management featuring Sign Up, Login, and Password Reset pages with secure session handling.
- AI Career Architect Tool: A generative AI tool that uses reasoning to analyze a user's professional background and interests to construct personalized long-term career roadmaps.
- Professional Networking Feed: A LinkedIn-inspired social hub where users share achievements, connect with mentors, and engage with industry-specific content.
- Skill-Based Course Marketplace: A robust catalog of educational content resembling Coursera and Udemy, enabling users to discover and enroll in specialized learning paths.
- Interactive Progress Dashboard: A centralized dashboard for tracking course completion, skill mastery, and professional milestones in a clean, visual interface.
- AI Skill Gap Analysis Tool: An intelligent tool that reasons through market trends and desired job roles to suggest specific learning modules to bridge the user's competency gaps.
- Protected Route Management: Security layer ensuring that personalized career data and learning progress are only accessible to authenticated users.

## Style Guidelines:

- Primary: Professional Navy (#1B365D) for authority; Secondary: Coursera Teal (#0056D2) for trust; Accent: Tech Purple (#573CED) for AI features.
- Headlines use 'Space Grotesk' for a modern, tech-forward look; Body text uses 'Inter' for high readability across long learning modules.
- A mix of professional networking glyphs and academic symbols, utilizing a consistent thin-line stroke style.
- A multi-column grid inspired by LinkedIn, featuring a central feed, sticky sidebar navigation, and a right-hand career progress tracker.
- Smooth micro-interactions for button hovers and soft fade-in transitions for page navigation to maintain a premium, high-end feel.
